# Header Component Specification

## Core Principle

`Header.tsx` renders the application title bar at the top of the page.

---

## Module Location

`src/ui/components/Header.tsx`

---

## Rendering

- Displays the text **"ModelLens"** aligned to the left.
- Background uses the `Surface` colour with a subtle bottom border (`Border` colour).
- Text uses `Text Primary` colour, bold weight, sized as a prominent heading.

---

## Props

None. This is a static presentational component.
